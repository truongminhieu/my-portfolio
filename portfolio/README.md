## Cách chạy local

1. Mở thư mục `portfolio`.
2. Mở tệp `index.html` bằng trình duyệt.
3. Website chạy trực tiếp mà không cần build hoặc cài đặt thêm.

## Cách deploy lên GitHub Pages

1. Tạo repository mới trên GitHub.
2. Upload toàn bộ thư mục `portfolio` lên repository.
3. Vào `Settings` > `Pages`.
4. Chọn branch `main` và thư mục gốc (`root`).
5. Lưu và lấy đường dẫn GitHub Pages.

> Không cần NodeJS, không cần build, không dùng React hay framework nào.
